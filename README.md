## Server Deployment

To run the server side code first,

```bash
  cd api
```

Install all packages and dependencies

```bash
  npm i
```

Run the server

```bash
  npm run dev
```

## Client Deployment

To run the client side code first,

```bash
  cd client
```

Install all packages and dependencies

```bash
  npm i
```

Run the client

```bash
  npm run dev
```
